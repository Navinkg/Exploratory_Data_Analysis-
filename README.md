🩺 Newborn Health Monitoring Analysis
📌 Project Overview

This project focuses on analyzing newborn health monitoring data to understand birth characteristics, vital signs, feeding patterns, and potential health risks. Through exploratory data analysis (EDA), the project aims to uncover patterns and insights that can assist healthcare professionals in monitoring infant well-being and identifying high-risk cases.

🎯 Objectives
Analyze newborn demographic and clinical information.
Examine birth measurements and growth indicators.
Identify patterns in vital signs and feeding behaviors.
Explore relationships between health parameters and risk levels.
Generate insights that support early health monitoring and decision-making.

📂 Dataset Information
The dataset contains records related to newborn health, including:
Baby ID
Gender
Date of Observation
Gestational Age
Birth Weight
Birth Length
Body Temperature
Heart Rate
Oxygen Saturation
Feeding Type
APGAR Score
Risk Level
Each record represents an individual newborn's health status at a particular point in time.

📈 Key Insights
Distribution patterns of birth weight and gestational age were examined.
Vital signs such as heart rate and oxygen saturation were analyzed.

High-risk cases can be identified through clinical indicators and health measurements.
