# 🩺 Newborn Health Monitoring Analysis

## 📖 Overview

This project performs Exploratory Data Analysis (EDA) on a newborn health monitoring dataset to analyze birth characteristics, vital signs, feeding patterns, and health risk levels. The analysis provides valuable insights into newborn health and supports data-driven healthcare decisions.

## 🎯 Objectives

* Clean and preprocess the dataset
* Perform Exploratory Data Analysis (EDA)
* Analyze newborn health indicators
* Identify patterns and correlations
* Generate meaningful visualizations

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Analysis Performed

* Data Cleaning
* Missing Value Analysis
* Statistical Summary
* Correlation Analysis
* Data Visualization
* Risk Level Analysis

## 📁 Dataset

The dataset contains information about newborn health, including:

* Gender
* Gestational Age
* Birth Weight
* Birth Length
* Heart Rate
* Oxygen Saturation
* Feeding Type
* APGAR Score
* Risk Level
